Character Guessing Game

Welcome to the character guessing game, where you guess the Touhou LostWord character by their silhouette!
There will be multiple silhouettes with the same names, as there are alternate versions of each character.

You have 5 guesses, guess the character before all of them are gone and you win! If not, better luck next time!

Costumes will also not be included in the silhouettes!
